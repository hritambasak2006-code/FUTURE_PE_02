# Prompt Engineering Workflow

## Prompt 1
Generate 10 UGC hooks for a modern cafe targeting students and young professionals.

## Prompt 2
Create a 30-second UGC advertisement script using Hook → Problem → Solution → Experience → CTA format.

## Prompt 3
Generate Instagram captions optimized for engagement and conversions.

## Prompt Engineering Insights

- Specific prompts produced more targeted outputs.
- Tone instructions improved authenticity.
- Structured prompts generated better marketing content.
- Iterative refinement improved engagement and clarity.

## Learning Outcome

This project demonstrates how prompt engineering can be used to create marketing assets such as hooks, ad scripts, captions, and CTAs using AI tools.
